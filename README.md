# Elo Rating Project

This project explores how the Elo rating system can be applied to different sports such as MMA, football, Formula 1, and Counter-Strike.

## Requirements

To run this project, you need:

- Python 3.10 or newer
- Jupyter Notebook or JupyterLab
- The following Python libraries:
  - pandas
  - matplotlib

## Installation

Install the required libraries with:

```bash
pip install pandas matplotlib notebook
